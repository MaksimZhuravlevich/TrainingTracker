Помощь

Как пользоваться FitTrack

Регистрация

  • Создайте аккаунт через «Регистрация» в шапке сайта
  • Укажите email, имя и пароль (минимум 8 символов)
  • После входа заполните физические данные при создании тренировки

Тренировки

  • Выберите спортивное направление
  • Укажите максимальные веса в упражнениях
  • Нажмите «Сгенерировать тренировку» — система подберёт нагрузку

Питание

  • Раздел «Питание» рассчитает калории и БЖУ
  • Данные подставляются из профиля автоматически

Прогресс

  • Смотрите историю тренировок в «Мои тренировки»
  • График прогресса доступен для каждого упражнения

Остались вопросы? Напишите на support@fittrack.ru
